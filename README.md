# bonter

A module for sharing Bahá'í stories with a touch of bonter.

## Installation

```bash
npm install bonter
```

## Usage

```javascript
const bonter = require('bonter');

// Tell a random Bahá'í story with a touch of bonter
bonter.tellRandomStory();
```
